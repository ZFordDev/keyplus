# keyplus

A completely offline, local-first password manager. 

Because putting all your keys in someone else's cloud is a bad idea, so I'm building my own.

## Core Pillars
* **100% Offline:** Local storage only. No sync servers, no cloud leaks.
* **Minimalist:** No bloat, just secure credential management.
* **Logic:** Built using the same core architecture as SchedPlus.
